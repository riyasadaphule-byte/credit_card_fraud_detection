# Credit Card Fraud Detection

## Overview
This project implements a supervised machine learning model to classify fraudulent and non-fraudulent credit card transactions.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Google Colab

## Methodology
- Data preprocessing and feature scaling
- Handling class imbalance
- Model training and evaluation using precision, recall, and ROC-AUC

## Results
The model achieves high recall for fraudulent transactions while maintaining low false positives.

## How to Run
Open the notebook in Google Colab and run all cells sequentially.
